# 🤖 AI RAG Chatbot using Streamlit, FAISS & Gemini AI

## 📌 Project Overview

This project is a Professional Retrieval-Augmented Generation (RAG) Chatbot developed using Streamlit, FAISS, HuggingFace Embeddings, and Google Gemini AI.

The chatbot is capable of answering user questions directly from PDF documents using semantic search and Generative AI techniques. The system retrieves relevant contextual information from uploaded PDF files and generates intelligent, context-aware responses in real time.

---

## 🚀 Features

* PDF-based Question Answering
* Retrieval-Augmented Generation (RAG)
* Semantic Search using FAISS Vector Database
* HuggingFace Sentence Transformer Embeddings
* Gemini AI-powered Response Generation
* Interactive Streamlit User Interface
* Source Chunk Visualization
* Real-time AI-powered Query Processing

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/rakeshvinothini/AI-RAG-Chatbot.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Setup Environment Variables

Create a `.env` file and add your Gemini API key:

```env
GOOGLE_API_KEY=your_api_key_here
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```
